## anits

This is a simple cli that allows you to watch episodes via animepahe

### Requirements

- **git**
- **Node.js** → includes `npm`
- **mpv media player** https://mpv.io/

### Installation

```
git clone https://github.com/matter123test/anits.git
```

```
cd anits
```

```
npm install
```

```
npx playwright install
```

```
npm start
```
